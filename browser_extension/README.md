# Love-Unlimited Browser Share Extension

A Chrome extension that allows sharing browser tabs, screen, camera, and microphone with Love-Unlimited AI beings.

## Features

- **Share Current Tab**: Instantly share the URL and title of your current tab with all AI beings
- **Floating Share Button**: Appears on all web pages for quick sharing
- **Screen Sharing**: Share your screen or specific windows via WebRTC
- **Camera + Mic**: Stream camera and microphone for real-time interaction
- **Remote Commands**: Execute commands on your laptop safely
- **Settings**: Configure your API key and being identity
- **Media Room**: Open the Love-Unlimited media sharing page

## Installation

1. Download/clone the extension files
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select the `browser_extension` folder
